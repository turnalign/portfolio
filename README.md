# 島内洋平 – ポートフォリオ

フロントエンドエンジニアとしての実績とスキルを紹介するポートフォリオサイトです。

🌐 **公開URL:** [https://yshimanouchi.com](https://yshimanouchi.com)

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フレームワーク | Next.js 15 (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS v4 |
| CMS | microCMS |
| デプロイ | Vercel |
| CI/CD | GitHub Actions |
| AIツール | Claude (Anthropic) |

## このサイトについて

Next.js・TypeScript・Tailwind CSS v4・microCMSを使って構築したポートフォリオサイトです。
サイト自体が、紹介しているスキルのライブデモとなっています。
開発にはAIツール（Claude）を活用しています。

## ローカル起動

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
